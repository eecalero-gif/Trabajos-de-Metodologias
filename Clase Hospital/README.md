# Hospital Management

Este proyecto tiene como objetivo ayudar al personal médico con cálculos básicos relacionados con el manejo de pacientes, como el cálculo de la dosis de medicamentos, el índice de masa corporal (IMC) y el registro de visitas.

## Métodos

- `calcularDosis(peso: number, mgPorKg: number): number`  
  Calcula la dosis total de medicamento en miligramos, dada la dosis por kilogramo.

- `calcularIMC(peso: number, altura: number): number`  
  Calcula el índice de masa corporal (IMC) de un paciente.

- `registrarVisitas(visitasActuales: number, nuevas: number): number`  
  Registra el número total de visitas de un paciente.
