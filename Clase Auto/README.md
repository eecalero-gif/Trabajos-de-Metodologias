# Simulador de Auto

Este proyecto simula el comportamiento básico de un auto, permitiendo acelerar, frenar y calcular la velocidad media en un trayecto.

## Requisitos

- Node.js
- TypeScript

## Instalación

1. Clona este repositorio.
2. Ejecuta `npm install` para instalar las dependencias.

## Uso

Para ejecutar las pruebas, usa el siguiente comando:

```bash
npm start
